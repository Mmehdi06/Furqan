Read the Quran the way it was meant to be read.

Furqan brings the authentic mushaf experience to your iPhone — beautifully rendered with the official **Quran Printing Complex (QPC)** fonts, **tajweed color coding**, and a clean, distraction-free interface. Every page mirrors the **Medina mushaf**, the most widely printed Quran in the world.

## Features

| Feature | Description |
|---|---|
| **Authentic Mushaf** | 604 pages matching the Medina print with QPC v4 color fonts |
| **Tajweed Colors** | Color-coded recitation rules embedded directly in the text |
| **4 Reading Themes** | Light, Dark, Sepia, and AMOLED — with full tajweed color preservation |
| **Smart Search** | Find any word in the Quran or jump to a verse reference (e.g. `23:65`) |
| **French Translation** | Complete Hamidullah translation for every verse |
| **Tafsir** | Ibn Kathir commentary for deeper understanding |
| **Bookmarks** | Save and organize your favorite verses |
| **Surah Index** | Navigate all 114 surahs with filtering by Makkah/Madinah |
| **Context Menu** | Long-press any ayah for translation, tafsir, surah info, or bookmark |
| **Animated Onboarding** | Beautiful 5-page onboarding introducing all features |

## Technical Highlights

- **CPAL/COLR font modification** — Dark mode fonts are pre-generated by modifying the color palette table (CPAL) in QPC v4 OpenType fonts at the binary level, turning black glyphs white while preserving tajweed colors
- **CGFont direct loading** — Dark font variants are loaded via `CGFont`/`CTFont` to avoid PostScript name collisions with light variants
- **Per-ayah context menu** — Uses Core Text `CTLine` hit-testing to detect which ayah was pressed, even on multi-ayah lines, with a custom lifted preview showing all lines of the ayah
- **Liquid Glass** — Native iOS 26 Liquid Glass support with graceful fallback for iOS 17-25
- **Performance optimized** — NSCache-based font caching, pre-computed line data, keyboard/search pre-warming during splash screen

## Architecture

```
Furqan/
├── FurqanApp.swift              # App entry, splash screen, onboarding
├── Models/
│   ├── QuranModels.swift        # Data models (SurahInfo, QuranPage, QuranLine, QuranWord)
│   └── ReadingTheme.swift       # Theme definitions and ThemeManager
├── Views/
│   ├── MushafPagerView.swift    # Main pager with toolbar navigation
│   ├── MushafPageView.swift     # Individual page layout
│   ├── MushafLineView.swift     # Line rendering (QPCLabel with context menu)
│   ├── SurahIndexView.swift     # Surah navigation with filters
│   ├── SearchView.swift         # Full-text search with verse reference lookup
│   ├── BookmarksView.swift      # Bookmark management
│   ├── TranslationView.swift    # French translation display
│   ├── TafsirView.swift         # Tafsir commentary display
│   ├── SurahInfoView.swift      # Surah background information
│   ├── ThemePickerView.swift    # Theme selection
│   ├── OnboardingView.swift     # Animated 5-page onboarding
│   └── AdaptiveGlass.swift      # Liquid Glass / material helpers
├── Extensions/
│   └── Font+Quran.swift         # QuranFontManager — font loading, dark font generation
├── Services/
│   ├── QuranDataService.swift   # SQLite data loading
│   ├── QuranSearchService.swift # Full-text search engine
│   ├── TafsirService.swift      # Tafsir data access
│   └── BookmarkManager.swift    # Bookmark persistence
└── Resources/
    ├── Fonts/                   # 604 QPC page fonts + surah header font
    └── Data/                    # SQLite databases (quran, search, translation, tafsir)
```

## Requirements

- iOS 17.0+
- iPhone only (portrait)
- Xcode 16+ / Swift 5.9+

## Privacy

Furqan does **not** collect any personal data. No analytics, no tracking, no accounts, no ads. All data (bookmarks, reading position, theme preference) is stored locally on-device only.
